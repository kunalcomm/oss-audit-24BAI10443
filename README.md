## Script Descriptions

### script1.sh — System Identity Report
Displays system info: distro, kernel, user, uptime, date and OS license.
Run: `bash script1.sh`

### script2.sh — FOSS Package Inspector
Checks if Python is installed, shows version and license, prints philosophy note.
Run: `bash script2.sh`

### script3.sh — Disk and Permission Auditor
Loops through key system directories and reports permissions and sizes.
Run: `bash script3.sh`

### script4.sh — Log File Analyzer
Reads a log file and counts lines matching a keyword.
Run: `bash script4.sh /var/log/dpkg.log install`

### script5.sh — Manifesto Generator
Asks 3 questions and generates a personal open source manifesto.
Run: `bash script5.sh`
